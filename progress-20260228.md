### Phase 6: NodeEval Type Instability Fix
- **Added** `FunctionWrappers.jl` dependency
- **Defined** `ActivationFn = FunctionWrapper{Float64, Tuple{Float64}}` and `AggregationFn = FunctionWrapper{Float64, Tuple{Vector{Float64}}}`
- **Rewrote** `NodeEval` type alias to use wrapped function types
- **Added** pre-allocated `_buffer::Vector{Float64}` to FeedForwardNetwork, RecurrentNetwork, CTRNNNetwork
- **Wrapped** activation/aggregation functions at construction time (once per genome lifetime, not per evaluation)
- **Rewrote** `activate!`/`advance!` inner loops to use buffer instead of allocating `Float64[]` per node
- **Updated** CTRNNNodeEval struct fields from `Function` to `ActivationFn`/`AggregationFn`
- **Fixed** 2 direct-constructor tests in runtests.jl to use new 5-arg constructor and wrapped functions
- **IZNN left unchanged** — uses Izhikevich equations directly, no Function type instability
- **Decision:** Registries (`ACTIVATION_FUNCTIONS`, `AGGREGATION_FUNCTIONS`) remain `Dict{Symbol, Function}` — wrapping happens at construction, preserving user-facing API

### Phase 7: Stagnation and Speciation Tests
- **Created** `test/test_stagnation.jl` (19 tests):
  - Basic stagnation detection (pre-populated fitness history)
  - No stagnation when improving (10-generation series)
  - Species elitism protection (3 species, elitism=2)
  - Single species with elitism can't go stagnant
  - Fitness function selection: max vs mean
- **Created** `test/test_speciation.jl` (17 tests):
  - Identical genomes → same species
  - Distant genomes → different species (low threshold)
  - GenomeDistanceCache correctness and cache hit verification
  - GenomeDistanceCache symmetry (one computation, both directions cached)
  - Species representative is a member
  - New species creation for distant genome
  - Genome distance function (hand-computed expected values)
- **Key insight:** `update!` resets `last_improved` on first call because `prev_fitness = -Inf`. Tests must pre-populate `fitness_history` to test stagnation without false negatives.
