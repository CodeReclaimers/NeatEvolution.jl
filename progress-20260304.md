# Progress Log — 2026-03-04

## neat-python 2.0.0 published on PyPI

**Date published:** March 2, 2026

neat-python version 2.0.0 was released to PyPI on March 2, 2026. This is the first major version bump since the 0.92 release in August 2017 (nearly 9 years). The 1.x series (1.0.0 on November 9, 2025; 1.1.0 on December 6, 2025) preceded it.

Key features in 2.0.0:
- Pure Python, no dependencies beyond stdlib
- Supports Python 3.8–3.14 and PyPy 3
- Reproducible evolution with random seed support
- Parallel fitness evaluation via multiprocessing
- Network export to JSON for framework interoperability

Package details: source tarball 126.0 kB, wheel 54.1 kB. Uploaded via twine/6.2.0 CPython/3.12.3.
