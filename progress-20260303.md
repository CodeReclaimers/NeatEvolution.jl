# Progress Log — 2026-03-03

## Renamed GitHub repository from NEAT to NeatEvolution.jl

**Summary:** Renamed the GitHub repository from `CodeReclaimers/NEAT` to `CodeReclaimers/NeatEvolution.jl` to follow the Julia package naming convention, where repository names include the `.jl` suffix while the package name in code does not.

**What changed:**
- GitHub repo renamed via `gh repo rename NeatEvolution.jl`
- Local git remote URL updated from `https://github.com/CodeReclaimers/NEAT.git` to `https://github.com/CodeReclaimers/NeatEvolution.jl.git`

**Why:** Julia's ecosystem convention (enforced by the General registry) is that repository names end in `.jl`. The package name in `Project.toml` and `using` statements remains `NeatEvolution` (no suffix). Aligning the repo name with this convention makes the project immediately recognizable as a Julia package.

**Note:** GitHub automatically redirects requests to the old `CodeReclaimers/NEAT` URL to the new name, so existing links and clones continue to work.
